Getting Started

  1. Install dependencies: 
    npm install
  2. Start the development server: 
    npm start
  3. Explore the project structure:
    • src/build/app.ts — Main entry point
    • modules/ — Your app's modules
    • config/modules.json — Module enable/disable config
  4. Create your first module: 
    // modules/hello/index.ts
export default {
  registerRoutes(app) {
    app.get('/hello', (req, res) => res.send('Hello from your first module!'));
  }
};
  5. Visit http://localhost:3000/ to see your app running!

For more advanced usage, see the Features and Architecture pages.
