Welcome to Nodesmith

Nodesmith is a modern, modular, event-driven Node.js framework designed for rapid development, scalability, and clean architecture.

Why Nodesmith?

  • 🚀 Auto-Discovery: Drop files in folders and they're automatically registered
  • 🏗️ Modular Architecture: Build large, maintainable apps with clear module boundaries
  • 🔗 Event-Driven: Decouple features using a powerful event bus
  • 💉 Dependency Injection: Leverage DI for testability and flexibility
  • ⚡ Zero Configuration: Automatic module loading and component registration
  • 📝 TypeScript-First: Strong typing and excellent developer experience

🎯 New: Auto-Discovery System

Simply drop files in the right folders and Nodesmith automatically discovers and registers:

  • routes/ - HTTP routes and controllers
  • events/ - Event handlers and listeners
  • services/ - Business logic and services
  • middleware/ - Custom middleware functions
  • types/ - TypeScript interfaces and types
