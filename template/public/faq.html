Frequently Asked Questions

What is Nodesmith?

Nodesmith is a modular, event-driven Node.js framework for building scalable, maintainable applications with TypeScript.

How do modules communicate?

Modules communicate exclusively via the Event Bus, ensuring loose coupling and extensibility.

How do I register a service?

Use the DI container to register and resolve services. See the DI page for examples.

How do I add a new module?

Add a folder in modules/ with a module.json manifest and an index.ts entry point. See the Modules page for details.

How do I handle errors in event handlers?

Emit *.failed events and listen for them to handle errors gracefully. See the Events page for patterns.

Can I use Nodesmith with JavaScript?

Nodesmith is TypeScript-first, but you can use it with JavaScript if you don't need type safety.

Where can I find more documentation?

Check the core/README.md in your project or explore the other pages of this site for in-depth guides.

How do I get help?

Open an issue on the project repository or join the community chat (link coming soon!).
